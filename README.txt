1.“线性规划.ipynb”在Anaconda上编写，需相同软件上运行。同名.py文件可以直接在Python3 IDE上运行。
2.本程序基于线性规划大M法编写。
3.功能：输入可以为不等式形式，目标函数可以最小或最大；可以准确判断唯一解，无界解，无穷解，无可行解情况，同时可以准确求解唯一解的线性规划问题以及无穷多解的一个解。
4.不足：线性规划输入变量强制要求必须为“X>=0”的形式。
5.测试样例见根目录同名文件，需要将其改名为“input.txt”并放置在与源代码相同的目录内，输出见“output.txt”。
6.提醒：程序输出的X不一定是按升序排列，因此记录结果时应特别注意。
7.程序输入必须同测试样例的形式，否则会出现不可预知的错误。
